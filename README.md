# 大纲编辑器

一个基于 Vue + TypeScript 的前端大纲编辑器，支持四级标题、拖拽排序、右键菜单、段落内容编辑及内容参考面板。

## 功能说明

### 标题层级与编号

支持四级标题，编号格式如下：

| 层级 | 格式 | 示例 |
|------|------|------|
| 一级 | 一、 | 一、测试概述 |
| 二级 | （一） | （一）测试背景与目的 |
| 三级 | N. | 1.核心概念界定 |
| 四级 | （N） | （1）样本分析定义 |

### 标题编辑

- 单击标题文本进入编辑模式，标题编号固定不可编辑，仅文本部分可输入
- 单击其他区域、按 Enter 或 Escape 退出编辑并保存

### 悬停交互

鼠标移动到标题上时：
- 显示灰色背景高亮
- 标题左侧显示拖拽标识（⠿）
- 标题右侧显示删除按钮（✕）

### 拖拽排序

- 拖拽任意标题可重新排列大纲结构
- 拖拽时显示被拖拽的内容，包含被拖拽标题及其所有子级，拖拽内容跟随移动效果
- 被拖拽的原位置显示为半透明状态
- 拖拽过程中显示蓝色箭头指示线，指示目标插入位置：
  - **上方箭头线**：插入为同级（前）
  - **下方箭头线**：插入为同级（后）
  - **缩进箭头线**：插入为子级（中间区域悬停触发，不超过四级限制）

### 右键菜单

右键点击标题弹出横向工具栏，包含以下操作（根据当前节点状态动态启用/禁用）：

| 操作 | 说明 |
|------|------|
| 设为段落内容 | 将叶子节点切换为段落模式（仅限无子节点时可用） |
| 设为段落标题 | 将段落节点切换回标题模式 |
| 添加同级 | 在当前节点后插入同层级新标题，默认标题展示为点击编辑提纲内容 |
| 添加下级 | 在当前节点下添加子标题（不超过四级），默认标题展示为点击编辑提纲内容|
| 复制 | 深度复制当前节点及其所有子节点 |
| 上移 | 与上一个同级节点交换位置 |
| 下移 | 与下一个同级节点交换位置 |

### 段落模式

- 叶子节点可通过右键菜单设置为"段落内容"
- 段落模式下移除编号前缀，文字样式有所区分
- 不允许在段落节点下添加子标题
- 可通过右键菜单"设为段落标题"切换回标题模式

### 段落内容编辑

- 叶子节点可附加段落内容区域
- 默认显示"点击编辑提纲内容"占位文字
- 单击后进入可编辑的文本框，内容为空；失焦后保存

### 内容参考面板

- 叶子节点右侧显示"内容参考"折叠按钮
- 单击展开面板，包含知识库 / 全部两个 Tab 切换
- 再次单击收起面板
